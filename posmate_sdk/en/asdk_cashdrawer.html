• Introduction
    We can open the cash drawer by writing GPIO. First, we output high to GPIO on the cash drawer, and second, wait for about 200 milliseconds, output low to GPIO.
     
  • Example
                   
//Example of cash drawer --
//This demo shows how to open cash draw by GPIO
private void mainCashDrawer(Device baseDevice){
    int CONFIG_CASHDRAWER_GPIO = (32*2+1);
    com.posmate.GPIO.setMode(baseDevice, CONFIG_CASHDRAWER_GPIO, 1, 0);
    	
    try {
        com.posmate.GPIO.output(baseDevice, CONFIG_CASHDRAWER_GPIO, 1);
        Thread.sleep(200);
        com.posmate.GPIO.output(baseDevice, CONFIG_CASHDRAWER_GPIO, 0);
        Thread.sleep(200);
    } catch (Exception ex) {
        ex.printStackTrace();
    }
}
