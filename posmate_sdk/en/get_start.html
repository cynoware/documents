• Get started
    Cynoware provides "PosMate SDK" for developers to build POS application quickly, which can be compiled on multiple platforms, such as windows and android.
    With "PosMate SDK", you can -
     
    - use COM port (UART) to control printer, LED display
    - use GPIO to open crash drawer
    - read QR code from QR reader
    - simulate hardware keyboard input
    - communicate with POS device without installing driver on windows
    - communicate with POS device without root on android
