• Introduction
    We can use UART to drive LED customer display. After configure the baudrate of LED's UART, we can show the data by writing the LED's UART instruction.
    (The following is for demo only. Other LEDs may have different baudrate and intructions.)
     
  • Example
                String[] ledtitle = new String[]{"price","total","collect","givechange","Init"};

public boolean msleep(long time){
    try {
        Thread.sleep(time);
    } catch (InterruptedException e) {
        return false;
    }
    return true;
}
