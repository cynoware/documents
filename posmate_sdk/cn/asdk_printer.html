• 简介
    可以使用UART驱动小票打印机，通常只要先设定好UART输出波特率， 然后按照小票打印机的指令文档，输出打印指令就可以了。
    （以下代码仅做示例，针对不同的打印机，波特率和指令系统可能不同。）
     
  • 示例代码
    public void mainPrinter(Device baseDevice){
    String printtxt = "Hello Cynoware!\n\n\n\n\n";

    String charsetname = "GBK";

    byte[] printtxtdata = printtxt.getBytes(Charset.forName(charsetname));

    int baudrate=115200;

    //int CONFIG_PRINTER_UART = 7;

    UART.setConfig(baseDevice, config.CONFIG_PRINTER_UART, baudrate, 8, UART.PRRITY_NONE, UART.STOPBITS_1);
    	
    UART.fixedWrite(baseDevice, config.CONFIG_PRINTER_UART, printtxtdata);//start to display text
}
